Any is like turning off the security system in TypeScript. It lets anything slide in or out, so the compiler stops protecting you. One wrong value and bugs sneak in quietly. unknown is the guarded version. It says “I don’t know what this is yet,” but forces you to check before using it. That checking is called type narrowing. Type narrowing means refining a broad type into a specific one using checks like typeof, instanceof, or custom conditions. Only after narrowing does TypeScript let you safely use the value. In short: any trusts blindly, unknown makes you prove it first.
